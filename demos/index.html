‚öôÔ∏è Technical Implementation

üîß For Developers

Multiple ways to implement border animations in your projects:

1. WordPress Shortcode (Legacy v1.0.0)

[border_animation type="rainbow" speed="2s"] Your content here [/border_animation]

2. CSS Classes (Direct Implementation)

<!-- Include the CSS file --> <link rel="stylesheet" href="border-animation-builder.css"> <!-- Apply animation classes --> <div class="border-animation-pulse"> <h2>Animated Content</h2> <p>This has a pulsing border animation</p> </div>

3. Custom CSS Variables

.my-animated-element { --border-color-1: #ff6b6b; --border-color-2: #4ecdc4; --animation-duration: 3s; --border-width: 3px; } .my-animated-element.border-animation-rainbow { /* Animation will use your custom variables */ }

4. JavaScript API (Advanced)

// Initialize animation builder const builder = new BorderAnimationBuilder(); // Create custom animation builder.createAnimation({ type: 'rainbow', colors: ['#ff6b6b', '#4ecdc4', '#45b7d1'], duration: '2s', direction: 'clockwise' }); // Apply to element builder.applyToElement('#my-element');
