A modern, lightweight plugin for animated border effects and utilities. Includes conic, solid, and dashed border animations, utility classes, and a demo shortcode.

== Description ==
* Add animated border effects to any block, widget, or template.
* Use utility classes or the [border_animation_demo] shortcode.
* Modern CSS (conic-gradient, mask, keyframes) with fallbacks.
* Accessible and responsive.

== Installation ==
1. Upload to the `/wp-content/plugins/` directory.
2. Activate through the 'Plugins' menu in WordPress.

== Usage ==
* Add classes like `ba-animate ba-conic` to your HTML.
* Or use `[border_animation_demo type="conic"]Content[/border_animation_demo]` in posts/pages.
